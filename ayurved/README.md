# OJAS Health

An Ayurveda-powered personalised health platform. OJAS helps users discover their Prakriti (body constitution) and receive tailored health guidance rooted in Ayurvedic principles.

## Repository Structure

```
/apps        — Application source code (Next.js 14 frontend, API services)
/packages    — Shared libraries, design system tokens, UI components
/docs        — Documentation, architecture decisions, project phases
/scripts     — Build, deployment, and utility scripts
/config      — Environment configs, infrastructure, CI/CD settings
```

## Tech Stack

- **Frontend**: Next.js 14 (App Router)
- **Database**: Supabase (PostgreSQL)
- **Styling**: TailwindCSS + custom design system tokens
- **AI**: Prakriti quiz logic engine + personalised recommendation system

## Getting Started

```bash
# Install dependencies
npm install

# Run development server
npm run dev
```
